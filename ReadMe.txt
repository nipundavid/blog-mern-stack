1. To Run -> npm run server
2. Register User -> http://localhost:5000/api/users
3. Auth User(use token from 2 in header x-auth-token) -> http://localhost:5000/api/auth
4. Get my profile (use token from 2 in header x-auth-token) -> http://localhost:5000/api/profile/me
5. Create & Update profile(use token from 2 in header x-auth-token) -> http://localhost:5000/api/profile
